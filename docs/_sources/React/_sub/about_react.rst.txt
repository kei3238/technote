============================================================
Reactについて
============================================================

`Reactとは <https://reactjs.org>`_
===============================================
* Facebookが開発・公開しているUIライブラリ.
* JavaScriptで構築されている.

Reactの特徴
===============

1. 仮想DOM(Document Object Model)
-----------------------------------------------------
リンクを踏んでページ遷移するのではなく、ページの構成要素を要素の中で
影響のあるところだけを再リレンダリング(描画)することで、動的にページを変更できる.
Facebookのいいねボタン(いいねするとボタン周辺だけが変化する)を想像するとイメージしやすい.

2. JSX
-----------------------------------------------------
JavaScriptを拡張した言語であるJSXを用いることにより、
処理(JavaScript)とページ構成(html)をシンプルに記述することができる.
JavaScriptのみでも記述は可能だが、可読性面からJSXの利用が推奨されている.

Reactを使うメリット
=======================
* サクサク動くページを作れる.
* 保守がしやすい.
* ReactNativeでスマホアプリも作れる.
