Express quick start¶

Expressの導入¶

  1. Project initialization

以下でプロジェクトを作成. とりあえず試すだけなら全てデフォルト回答でOK.

$ npm init

  1. expressのインストール
npm install --save express

Express quick start¶

  1. エントリーポイントになるjsファイルを作成
touch index.js
  1. index.jsを以下のように編集
var http = require('http');
var express = require('express');

var app = express();

app.use(function(req, res, next) {
  return res.send('Hello express!');
});

var server = http.createServer(app);
server.listen('3000');
  1. Expressを起動
node index.js
  1. 適当なブラウザで http://localhost:3000 にアクセスして"Hello express!"と表示されることを確認.
