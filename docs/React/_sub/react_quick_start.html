Quick Start¶

Nodeのインストール¶

  1. node.jsからダウンロード
  2. ターミナルでバージョンを確認できればOK
$ node -v
v9.7.1
  1. npmのバージョンも確認
$ npm -v
5.6.0

create-react-appでプロジェクト作成¶

  1. npmでcreate-react-appをインストール
npm install -g create-react-app
  1. create-react-appを実行
create-react-app sample-app
  1. sample-app内に以下のようなファイル&ディレクトリが作成される
.
├── README.md
├── node_modules
├── package-lock.json
├── package.json
├── public
└── src
  1. sample-appディレクトリに移動してnpm start
cd sample-app
npm start
  1. http://localhost:3000 でsample-appが実行される(自動でブラウザが開く)
